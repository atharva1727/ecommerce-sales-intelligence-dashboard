# E-Commerce Sales Intelligence Dashboard

## Overview
This project presents an interactive Business Intelligence dashboard developed using Power BI to analyze E-Commerce sales performance and provide actionable insights.

## Objectives
- Track order profitability
- Identify high revenue markets
- Analyze discount impact
- Monitor customer buying patterns
- Improve inventory performance

## Features
✔ Profitability Analysis  
✔ Revenue Tracking by Region  
✔ Discount Impact Visualization  
✔ KPI Dashboard  
✔ Customer Trend Analysis  
✔ Sales Growth Monitoring  

## Technologies Used
- Power BI
- Excel / CSV
- ETL
- Data Visualization

## Dashboard Insights
- High performing regions identified
- Customer buying trends analyzed
- Revenue and profit patterns tracked
- Discount losses minimized
